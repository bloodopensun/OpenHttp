# OpenHttp
由于工作原因以及个人爱好,经常需要抓取一些网络资源和调用一些非公开接口,使用了很多.Net自身组件,但并不太好用

并且本人对socket开发并不怎么熟悉,故只能在HttpWebRequest和HttpWebResponse做一些封装,使他们更好用(对我来说)

## 功能概述

- 轻量级(一个文件,上千行代码)
- 用于简单的Http/Https请求
- 使用链式的方式组装请求Head
- 自动识别压缩方式,自动解压
- 暂时只支持Get/Post请求
